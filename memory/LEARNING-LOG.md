# Learning Log

Self-improvement feedback loop for the pre-market routine. Each pre-market run scores
**yesterday's** research against yesterday's **actual** results (top tradeable
gainers/losers from Alpaca), names the best trade it realistically could have proposed,
and distills lessons. The pre-market routine reads the last ~7 scorecards to softly bias
today's focus toward recurring winning sectors and away from repeated-miss patterns.

Soft bias only — these lessons NEVER override hard rules (3% risk, quant ≥3/5, 12% heat,
position/sleeve caps, regime gate). Newest entry on top.

Filtered/tradeable universe = price ≥ $5, no warrants/units/rights.
